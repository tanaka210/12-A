フォーム送信の違いを体験しよう

① GETで送信する

送信後のURLを確認する。入力したデータがURLに表示される。

コメント（name属性なし）

→ name属性がないフィールドはURLに出てこない（サーバーにデータが届かない）

送信後のURL例：3-5_thanks.html?name=田中&email=tanaka%40example.com
name属性の値がURLのキー名になる。name属性のないフィールドはURLに現れない。

② POSTで送信する

送信後のURLを確認する。データはURLに表示されない。開発者ツール（F12 → Network → Payload）で確認できる。

送信後のURL：3-5_thanks.html（データはURLに出ない）
開発者ツール → Network → リクエストを選択 → Payload タブでデータを確認できる。
